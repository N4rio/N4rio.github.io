🐘 DouShou Qi — Mise à jour 2.0.0

Voici les dernières nouveautés et améliorations apportées à Dou Shou Qi.
Encore merci pour vos retours et votre soutien ! 🙏

  • Personnalisation des noms : modifiez votre nom et celui de votre adversaire directement depuis les paramètres ⚙️
  • Refonte graphique complète : une interface modernisée avec de nouveaux boutons, images et effets visuels 🎨
  • Leaderboard amélioré : le temps de chaque partie est désormais affiché ⏱️
  • Calcul du score ajusté : la fin de partie favorise désormais les stratégies intelligentes plutôt que les simples éliminations 🧠
  • Nouveau design des pièces : les animaux sont plus visibles et facilement reconnaissables 🐯🐭
  • Abandon possible : vous pouvez maintenant abandonner une partie en cours si nécessaire 🏳️
  • Indication du joueur actif : une couleur claire indique à qui c'est de jouer 🎯
  • Modifications dans les paramètres :
    • 🌐 La langue du jeu est maintenant synchronisée avec la langue de votre téléphone
    • 💧 Le mode Aquatique et le mode Informatique ont été retirés (contenu insuffisant)
  • 🆕 Nouveaux modes de jeu :
    • Mode Requin : une nouvelle pièce fait son apparition ! Le requin nage uniquement dans les rivières et adore pourchasser les rats imprudents 🦈
    • Mode Anarchie : chaque partie devient unique ! Les pièces sont placées aléatoirement (mais symétriquement) sur votre côté du plateau 🎲
  • 🐞 Corrections : de nombreux bugs mineurs ont été corrigés pour une expérience plus fluide
